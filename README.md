# Mineverso_3D
El proyecto propone el desarrollo de DataVerso Minero, una iniciativa formativa y tecnológica dirigida principalmente a estudiantes de Ingeniería de Minas, quienes serán preparados como Agentes de Modelado Digital Minero capaces de construir entornos 3D, recorridos virtuales y espacios digitales medibles de minas subterráneas y superficiales. La propuesta combina realidad virtual, realidad aumentada, modelado 3D y medición técnica para que los futuros profesionales no solo visualicen una mina, sino que aprendan a representar, medir y analizar datos como distancias, anchos, alturas, rutas, áreas, coordenadas y zonas críticas.
La necesidad surge porque muchos estudiantes reciben una formación principalmente teórica y tienen pocas oportunidades de interactuar con escenarios mineros reales o digitales aplicados. Al mismo tiempo, las empresas mineras requieren personal con competencias digitales que pueda apoyar procesos de capacitación, inducción, planeamiento, seguridad y documentación técnica.
